¿Qué es github? Es una página de internet que se usa para proyectos de programación, es una extensión de GIT. GIT es la nube y github es la página de internet donde podemos ver los cambios. La principal función de github es hacer trabajos de programación con otras personas, ellos se velven colaboradores de ese proyecto, y cada quien va haciendo sus cambios.
